# To-Do-List

  The **Advanced To-Do List Application**, a modern, feature-rich task management system crafted with **HTML, CSS, and JavaScript**. This intuitive application enables users to **add, complete, and delete tasks**, each configurable with a **title, category, priority, and due date**.
  
  Beyond basic functionality, it offers powerful features like **advanced filtering** (by status, category, and overdue tasks), **visual indicators for priority levels** (High, Medium, Low), and smart due date formatting. The app provides **real-time statistics** on total, completed, and pending tasks, all within a **responsive design** that adapts seamlessly to various screen sizes. Built around a `TodoApp` class, it ensures a structured codebase and is prepared for **persistent data storage** for an enhanced user experience.
